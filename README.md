Website yang dibuat merupakan website sederhana menggunakan HTML dan CSS yang menampilkan daftar café di Yogyakarta. Website ini terdiri dari bagian header, isi (card café), dan footer.

Setiap café ditampilkan dalam bentuk card yang berisi gambar, nama café, dan link menuju lokasi di Google Maps.
